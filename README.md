Baja Telemetria App — Protótipo v2

Equipe Intel — MudRunner Baja — CEFET/RJ

Este repositório contém a segunda versão do protótipo do Baja Telemetria App, desenvolvido para a equipe Intel do projeto MudRunner Baja do CEFET/RJ.

O objetivo deste aplicativo é permitir a visualização rápida e intuitiva de dados do veículo, facilitando o acompanhamento em tempo real durante testes e competições.

Nesta versão:

O app foi desenvolvido utilizando Android Studio junto com Kotlin, com foco na experiência mobile.

O protótipo está otimizado para desempenho e usabilidade.

Futuras atualizações planejam:

Integração de gráficos interativos para análise dos dados do veículo.

Possibilidade de gerenciamento e exportação de arquivos diretamente pelo app.
